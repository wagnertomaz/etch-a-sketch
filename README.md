# 🎨 Etch-a-Sketch

Projeto de **Etch-a-Sketch** para navegador, desenvolvido como parte do curso **The Odin Project**.  
O objetivo é recriar o brinquedo de desenho usando tecnologias web.

## 📚 Sobre o Projeto

O Etch-a-Sketch permite:
- Desenhar passando o mouse sobre a grade  
- Personalizar o **modo de cores**  
- Ajustar o **tamanho da grade** para mais ou menos pixels  

Este projeto é uma ótima prática para consolidar conhecimentos em HTML, CSS e JavaScript, além de manipulação de DOM.

---

## 🚀 Tecnologias Utilizadas

- **HTML5**  
- **CSS3**  
- **JavaScript**  
  - Usado para detectar movimentos do mouse, atualizar cores e gerar a grade dinâmica.
